## 1. Added Gate Fields

新增字段写入 M1.3 candidate comparison 的 per-query `baselineProtection`，只包含 rank/count/label/reason code：

| Field | Definition | Source |
| --- | --- | --- |
| `baseline_best_relevant_rank` | baseline rows 中 qrels relevance `>=2` 的最佳 rank | `rows_by_mode["baseline"]` + product qrels |
| `candidate_best_relevant_rank` | 当前 candidate rows 中 qrels relevance `>=2` 的最佳 rank | candidate mode rows + product qrels |
| `rank_delta_vs_baseline` | `candidate_best_relevant_rank - baseline_best_relevant_rank`；rank 不可比时为 `null` | runner 派生 |
| `metric_regression_reason` | after-vs-baseline regression 的脱敏 reason code；无退化为 `NONE` | label、tags、best relevant rank 派生 |

这些字段用于证明 baseline 成功样本是否被 candidate 压低。Aggregate Top10 只能说明总体命中率，不能覆盖单 query 的 Top10 hit lost、NDCG/P@5 下降或 baseline relevant rank 后移。
