## 二、实现概述（纯前端 + 元数据 only，后端 0 文件改动）

复用 M4-1 已冻结 `ENABLE_REPORT_TEMPLATE=false`；报告导出复用 M4-5 下载能力。后端无任何改动，沿用 M4-5 末态。

- 新增 `apps/web/src/lib/reportTemplate.ts`（纯函数 + 可注入下载器）：
  - 数据结构 `ReportTemplate`：`report_id` / `list_id` / `report_title` / `generated_at` / `report_status` / `degrade_reason` / `item_count` / `sections[]`。
  - 5 类章节 `ReportSection`：`search_background`（检索背景占位 + 用户自填短备注）/ `list_overview`（清单概览）/ `case_entries`（逐案元数据 + 来源锚点 + 用户自填 + 已守门 AI 片段）/ `review_points`（待人工复核要点，中性动作清单）/ `disclaimer`（强制免责）。
  - 逐案条目 `ReportCaseEntry` 字段白名单：ordinal / case_id / case_number / court / trial_level / case_cause / judgment_date / source_anchors / note / tag / anchored_fragments，**零正文**。
  - `buildReportTemplate`：组装报告；缺清单→failed/list_not_found，空清单→failed/empty_list，组装异常→degraded/assembly_failed（仅清单概览 + 免责）。全程不抛出。
  - 锚点与 AI 片段守门：`sanitizeReportAnchors`（只留 case_id+source_chunk_id 齐全、归属本案的锚点）；`filterAnchoredFragments`（片段必须①归属清单内案例②携带完整来源锚点且锚点 case_id 与片段一致③文本非空、截断到 160 字、不含禁用话术；任一不满足即丢弃）。默认无片段→纯元数据骨架。
  - `renderReportMarkdown`：渲染为 Markdown（预览 / 导出共用），仅结构化字段；AI 片段渲染时强制带 `[来源：case_id#chunk_id]`。
  - `downloadReport`：复用 M4-5 `getBrowserDownloader` 注入式下载器，仅本地生成 `.md` 文件；failed 报告不下载；无下载器→degraded/download_unavailable（仍返回内容供预览）；下载器抛错→degraded/download_failed。
  - 脱敏日志 `report_template_action`：仅 event / status / reason_code / section_count / item_count，无正文 / 案号 / note / tag / title / query。
- 新增 `apps/web/src/components/results/ReportPreview.tsx`：报告预览组件，按章节渲染；检索背景为用户自填短备注（≤240 字，系统不自动生成案情或结论）；降级态给安全提示；组件不组装内容、不读写 storage、不发送任何正文。
- 接线 `CaseListPanel.tsx`（reportEnabled + onGenerateReport + onDownloadReport，清单非空 + flag 开才渲染「生成类案报告模板」入口）+ `SearchPage.tsx`（handleGenerateReport / handleDownloadReport，flag 关闭返回 failed 安全态）。

## 四、内容边界与免责（验收核心）

- 报告只含：模板结构 + 元数据 + 来源锚点 + 用户自填短字段（检索背景备注 / note / tag）+ 系统生成的结构化占位（待复核要点）+ 强制免责说明。
- **不自动起草**代理词 / 诉状 / 答辩状 / 庭审提纲；**不输出**胜诉 / 败诉概率或确定性法律结论 / 确定性论证。
- AI 加工片段 **100% 携带来源锚点**（case_id+source_chunk_id），无锚点 / 跨案 / 含禁用话术一律丢弃，不进入报告。
- 强制免责 4 行：仅元数据不含正文；阶段性结果可能未覆盖、不承诺完整检索；不提供胜负判断、不构成确定性法律结论 / 法律意见、不可直接作法律文书；须律师结合权威库与原文人工复核。
- 运行时自检 `reportRenderHasForbiddenPhrase` 复用 M4-5 禁用词表（覆盖胜诉率 / 查全率等 11 词）。
